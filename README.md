# Bank Customer Churn Prediction

## Overview

This project predicts customer churn using machine learning and explainable AI techniques.

## Models Used

* Logistic Regression
* Decision Tree
* Random Forest
* Gradient Boosting (Best Model)

## Results

* Accuracy: 86.9%
* ROC-AUC: 0.868

## Deployment

Streamlit dashboard is used for real-time prediction.
